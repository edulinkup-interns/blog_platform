# 🚀 Blog Platform

A full-stack blogging application built using the MERN stack. This platform allows users to create, explore, and engage with blogs through a clean and modern interface inspired by social media platforms.

---

## ✨ Features

- 🔐 User Authentication (JWT)
- ✍️ Rich Text Editor (TinyMCE)
- ❤️ Like & 💬 Comment System
- 🔍 Search Functionality
- 🏷️ Category Filtering
- 📄 Pagination
- 🖼️ Image Upload Support
- 📱 Fully Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS (Custom Styling)

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2️⃣ Setup Backend
cd backend
npm install

Create a .env file in the backend folder:

MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5001

Run backend:

npm start
3️⃣ Setup Frontend
cd frontend
npm install
npm start
